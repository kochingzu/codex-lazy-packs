# 專案初始化（Codex 版）

1. 問：名稱、用途、資料夾、GitHub repo、公開/私有、部署需求
2. 建立：AGENTS.md、README.md、.gitignore、Git repo、GitHub repo、Obsidian 工作筆記
3. 安裝開工/收工 skills：`npx skills add mathruffian-dot/opencode-lazy-packs --skill 07-workflow-skills -g -y`

## 開工/收工
開工：讀 AGENTS.md → 讀 Obsidian 筆記 → git status → 回報
收工：檢查敏感資料 → 更新筆記 → git commit/push → chezmoi 同步（見 13-chezmoi）

回報：檔案清單、GitHub URL、Obsidian 筆記路徑。
